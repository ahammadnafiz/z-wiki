# Raw Sources

Drop source material here. Claude never modifies this directory.

Subfolders:

- `articles/` — blog posts, newsletters, scraped web pages, X/Twitter long-forms, markdown notes
- `papers/` — PDFs and long-form research (academic papers, theses, reports). Uses the paper-summary template on ingest.
- `transcripts/` — YouTube, podcast, meeting transcripts
- `assets/` — images referenced by markdown sources (Obsidian's default paste target)

Filename convention for `raw/` files: anything reasonable works. Claude will derive a kebab-case slug for the wiki summary. Put the URL at the top of markdown articles (`Source: <url>`) so Claude can cite it.
